## ¿Qué tiene?

- Operaciones básicas: suma, resta, multiplicación y división
- Cálculos encadenados con la expresión completa visible (ej: `3 + 5 + 2 = 10`)
- Historial de operaciones con scroll
- Reutilizar cualquier resultado del historial
- Borrar entradas individuales del historial
- Panel del historial que se abre y cierra con animación

---

## Cómo correrlo

```bash
git clone https://github.com/felipo28/calculadora.git
cd calculadora
npm install
ng serve
```

Luego abre `http://localhost:4200` en el navegador.
